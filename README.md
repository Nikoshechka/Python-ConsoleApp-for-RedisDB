# Консольный клиент Redis

Простой консольный клиент для взаимодействия с сервером Redis.

▌Описание

Этот скрипт на Python предоставляет базовый интерфейс командной строки для установки и получения пар ключ-значение в базе данных Redis. Он обрабатывает подключение к серверу Redis, пользовательский ввод и кодирование/декодирование данных.

▌Использование

1.  Предварительные требования:
   
    •   Python 3.x
    
    •   Запущенный сервер Redis (например, на localhost:6379)
    
    •   Библиотека redis-py: pip install redis
    

3.  Запуск скрипта:

    python index.py
    

3.  Взаимодействие с клиентом:

    •   Следуйте подсказкам меню на экране, чтобы выбрать действие:
        *   1: Установить пару ключ-значение.
        *   2: Получить значение по заданному ключу.
        *   3: Выйти из клиента.

▌Конфигурация

Следующие параметры подключения можно настроить в начале скрипта:

•   REDIS_HOST: Хост сервера Redis (по умолчанию: 'localhost').
•   REDIS_PORT: Порт сервера Redis (по умолчанию: 6379).
•   REDIS_DB: Номер базы данных Redis (по умолчанию: 0).

▌Пример

```
Консольный Redis Клиент
==============================
1. Добавить/Обновить данные (SET)
2. Получить данные (GET)
3. Выйти
==============================
Выберите опцию (1-3): 1
Введите ключ (Key): mykey
Введите значение (Value): myvalue

[УСПЕХ] Установлено: 'mykey' = 'myvalue'
```
